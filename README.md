# Fibonacci Slice Generator
This is a test project

Usage
------------

First you need to install docker, you can take it by this command:

``` {.sourceCode .bash}
$ ./run.sh setup
```

And then tun project
``` {.sourceCode .bash}
$ ./run.sh stack
```
Running tests:
``` {.sourceCode .bash}
$ ./run.sh test style
$ ./run.sh test unit
$ ./run.sh test functional
```
